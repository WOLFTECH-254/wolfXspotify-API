## Getting Started

### Prerequisites
- Node.js 18 or higher
- npm 8 or higher
- Git

### Local Setup

```bash
git clone https://github.com/WOLFTECH-254/wolfXspotify-API
cd wolf-track-api
npm install
cp .env.example .env
npm run dev
